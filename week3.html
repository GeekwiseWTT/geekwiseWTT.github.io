What is Javascript?

Created in 1995 by Brendan Eich of Netscape in 10 days.

Started out named Mocha, changed to LiveScript and again to JavaScript.

The rendering technology is embedded into every web browser.

Developed so programmers could make engaging and dynamic websites that run in the browser, not on the server.

Reserved Words

There are some words, sometimes called keywords that are reserved in the JavaScript language. These words have special meaning and functionality. Here is a short list of the ones that we’ll come across:

Most Common if/else if/else loops switch statements Other useful keywords
var if for switch let
function else if while case const
return else do continue typeof
this in break

Tip: You probably won’t find name in any list of reserved words. However, it IS utilized by the window object, so don’t use it. It'll cause problems.

The Console

This is where we will do all of our debugging. If an error is "thrown", you'll see it here. To access the console, right click anywhere on the browser window and select "Inspect element" then select the "Console" tab.

Methods:

.log()

console.log('Message in the console.');

Displays a simple message in the console and includes a link to the source of the message.

.info()

console.info('Info message in the console.');

Displays a message with an information icon and includes a link to the source of the message.

.warn()

console.warn('Warning message in the console.');

Displays a message with a warning icon and includes a link to the source of the message.

.error()

console.error('Error message in the console.');

Displays a message with an error icon and includes a link to the source of the message.

.dir()

console.dir([4, false, 'Ian Malcolm']);

Displays an interactive list of an object's properties.

Alert / Confirm / Prompt

alert()

alert("This is an alert! Click ok to continue.");

The alert() method displays an alert box with a specified message and an OK button.

An alert box is often used if you want to make sure information comes through to the user.

confirm()

confirm("Here's an interesting offer. Do you say OK and take it, or Cancel the offer and run away.");

The confirm() method displays a dialog box with a specified message, along with an OK and a Cancel button.

A confirm box is often used if you want the user to verify or accept something.

The confirm() returns a true or false value that can be captured based on the users selection.

var answer = confirm("Okay or Cancel?");
console.log(answer);

prompt()

The prompt() method displays a dialog box that prompts the visitor for input.

A prompt box is often used if you want the user to input a value before entering a page.

prompt("What is your name?");

The prompt() returns whatever the user types into the text box.

Note: For the prompt(), the returned value will always be a string.

var username = prompt("What is your name?");
console.log(username);
