Practice

Steps:

  • 1. From your own Github account, create a new repository and clone to your laptop
  • 2. Set up the basic file structure for your project and push to create your master branch (include bootstrap)
  • 3. Create a new branch called home-page and create a home page using bootstrap components. (navbar, jumbotron/carousel, content and footer)
  • 4. To complete your assignment: add, commit, and push your code.
  • 5. Add me as a contributor (purplyrocky) and create a Pull Request. Make sure you tag me in the PR.
  • 6. Once I approve your PR, merge and close your pull request
  • 7. Delete your practice branch both locally and on github.com (optional)
