Arrays

The JavaScript Array object is a global object that is used in the construction of arrays; which are list-like objects. Arrays are used to store multiple values in a single variable. Arrays can also store different types of elements. You can put objects, entire functions or other arrays into arrays!

Arrays are zero based and every item has a corresponding index.

Note: The forEach() loop is a special type of loop used exclusively for Arrays. It offers a lot of options for accessing the contents of an Array. There are many other types of loops that we'll be exploring soon.

Note: Inside of the forEach() is an anonymous function. This function —as shown between the perenthesis— is expecting two things: an item and an index. These "expectations" are called parameters, and are being provided by the forEach() loop.

Array Methods

.sort()

The method sorts the elements of an array in place and returns the array. The default sort order is according to string Unicode code points.

Here's a breakdown of the calculation:

2, 4, 5, 1, 3 — 2 is less than 4, move to the next two.

2, 4, 5, 1, 3 — 4 is less than 5, move to the next two.

2, 4, 5, 1, 3 — 5 is greater than 1... flip the numbers and work backward.

2, 4, 1, 5, 3 — 4 is greater than 1... flip the numbers and work backward.

2, 1, 4, 5, 3 — 2 is greater than 1... flip the numbers. Continue from the index that began the direction shift.

1, 2, 4, 5, 3 — 5 is greater than 3... flip the numbers and work backward.

1, 2, 4, 3, 5 — 4 is greater than 3... flip the numbers and work backward.

1, 2, 3, 4, 5 — 2 is less than 3... sorting complete.

Loops

The for statement creates a loop that consists of three optional expressions, enclosed in parentheses and separated by semicolons, followed by a statement (usually a block statement) to be executed in the loop.

Loops are handy, if you want to run the same code over and over again, each time with a different value.

Comparison & Logical Operators

Comparison Operators

  • Equal to
    • ==
  • Strict equal to (compares both value AND type)
    • ===
  • Not equal to
    • !=
  • Strict not equal to (not value OR type)
    • !==
  • Greater than
    • >
  • Less than
    • <
  • Greater than or equal to
    • >=
  • Less than or equal to
    • <=

Logical Operators

  • And
    • &&
  • Or
    • ||
  • Not
    • !

If, Else If & Else

The if statement executes a statement if a specified condition is truthy. If the condition is falsy, another statement can be executed.
