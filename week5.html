DOM Manipulation

You can change any HTML using Javascript. When you do so, you're manipulating the DOM. You can learn more about the DOM from MDN and from w3schools.

Forms

Forms are a pretty complex beast. Accessing the user content can be potentially complicated and done in a number of ways. We'll look at a simple form and a complex form.

This

A function's this keyword behaves a little differently in JavaScript compared to other languages. In most cases, the value of this is determined by how a function is called. It can't be set by assignment during execution, and it may be different each time the function is called.

Creating Elements

Objects

Virtually everything in JavaScript is an Object. An Object can hold many different things, but it starts with a variable. Just like a variable is a container, an object is a variable as well except it can hold many different values. Each item that's saved in an object is known as a name:value pair.

Object Constructor

The object constructor gives us a skeleton of an object that can be used over and over. Each usage creates a new instance of the object
